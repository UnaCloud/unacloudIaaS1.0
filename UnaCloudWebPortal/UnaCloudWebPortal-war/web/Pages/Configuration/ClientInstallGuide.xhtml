Windows instructions:

Download the and copy them on UnaCloud_Home directory on all the machines that are going to partipate on UnaCloud deployments.
After copy is done, edit each one of the scripts on UnaCloud_Home and replace the first line cd C:\agente\unacloud\ for cd UnaCloud_Home using your UnaCloud directory path.
Now choose one of the next steps according to your infrastructure configuration:

Option 1: Local Group Policies

1. Run gpedit.msc on command line or the Run... window (Windows Home + R or Start Menu -> Run...).
2. Open Machine Configuration/Windows configuration/Scripts.
3. Double click on start and add a new script that points to UnaCloud_Home/start.bat with no parameters.
4. Return to Machine Configuration/Windows configuration/Scripts.
5. Double click on Shutdown and add a new script that points to UnaCloud_Home/stop.bat with no parameters.
6. Open User Configuration/Windows configuration/Scripts.
7. Double click on login and add a new script that points to UnaCloud_Home/logon.bat with no parameters.
6. Return to User Configuration/Windows configuration/Scripts.
9. Double click on logoff and add a new script that points to UnaCloud_Home/logoff.bat with no parameters.

Option 2: Domain Group Policies

Do Option 1 but using the Domain Group Policy Manager instead.
